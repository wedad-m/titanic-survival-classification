# Titanic Survival Classification

This project focuses on predicting passenger survival on the Titanic using supervised machine learning techniques.

## Project Overview
The workflow includes loading and exploring the dataset, preprocessing the data, splitting it into training and testing sets, training a classification model, and evaluating its performance.

## Contents
- Titanic dataset (`train.csv`)
- Jupyter Notebook for data exploration (`titanic_data_exploration.ipynb`)
- Machine learning methodology diagram (`ml_methodology_diagram.png`)
